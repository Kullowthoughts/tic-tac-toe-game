# Tic-Tac-Toe

A simple command-line or web-based Tic-Tac-Toe game built with JavaScript, HTML, and CSS. Players take turns marking spaces in a 3Ã—3 grid, aiming to get three of their marks in a row, column, or diagonal to win.

---

## Features

- Two-player gameplay (X and O)
- Highlights the winning combination
- Detects draws
- Responsive design (if web version)
- Easy to reset and play again

---

## Demo

If hosted on GitHub Pages or locally, you can run the game in your browser.

---

## Installation

1. Clone the repository:

```bash
git clone git@github.com:Kullowthoughts/tic-tac-toe-game.git
````

2. Navigate to the project folder:

```bash
cd tic-tac-toe-game
```

3. Open `index.html` in your browser.

---

## Usage

* Player X starts first.
* Click or select a square to mark it.
* The game will announce the winner or a draw.
* Use the reset button to start a new game.
